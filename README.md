## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Nihan-k/nihank_horizon_s2.git
   ```
2. Navigate to the project directory:
   ```bash
   cd nihank_horizon_s2
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
